# Introduction
This is a online resume project, it uses HTML, CSS and JavaScript.

# Structure
- Homepage: is built in index.html and index.css is used to decorate it.
- Project page: is built in projects.html and projects.css is used to decorate it.
- Other pages:is built in the .html file whose name is same as the name of the page.
  style.css is used to decorate all of these pages.
